# Attendance Processor Application

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/theblues88/my-calculator.git
   cd my-calculator
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the application:
   ```bash
   npm start
   ```
2. Follow the on-screen instructions to input attendance data.
3. To process the attendance, use the following command:
   ```bash
   npm run process-attendance
   ```
4. For more options, check the help command:
   ```bash
   npm run help
   ```

## Note
Ensure you have Node.js and npm installed on your machine before running the application.
